# 项目概述 java1078

本项目是一款基于Springboot和Vue的前后端分离的排课管理系统，旨在为中小学教师提供灵活、便捷的课程安排服务。

## 技术栈

- 后端：Java、SpringBoot
- 前端：Vue
- 数据库：MySQL

## 功能模块

- 课程档案模块
- 课位档案模块
- 排课申请模块
- 排课结果模块
- 用户管理模块
- 部门管理模块
- 角色管理模块
- 菜单管理模块
- 日志管理模块
- 数据字典管理模块
- 文件管理模块
- 图表展示模块

## 系统角色

系统基于角色的访问控制，主要面向教务管理员和教师角色，权限可以精确到按钮级别。

## 运行环境

- 后端：Java 1.8+、SpringBoot 2.x
- 前端：Vue 2.x
- 数据库：MySQL 5.7+

## 部署步骤

1. 准备Java和MySQL运行环境
2. 导入数据库文件
3. 部署后端服务
4. 部署前端服务
